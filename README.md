# SD.Next Canvas

## Install and Run

> npm install  
> npm run dev  
> navigate to: <http://localhost:8000/index.html>  

## Working

- load initial image and set stage size
- load multi-image
- move/resize/rotate image
- expand stage: drag or resize image outside of the stage
- compress stage: double click on empty area
